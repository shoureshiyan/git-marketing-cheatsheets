# git-marketing-cheatsheets
cheatsheets für Git und Markdown
Auf profil gehen
Klicke auf **repoitory**
Gib einen Namen ein,z.b:**git-markdow-cheatsheets**
Wähle public
Aktiviere **Add README file**
Klicke **Create repository** 
Schuche im Internet z.b. nach: **Git Cheat sheet PDF** oder **Markdown Cheat sheet**
Öffne dein Repository
Klicke **Add file**
Klicke **Upload file**
Lade die Cheatsheets hoch (PDF oder .md Datei)
Unten schreibe bei Commit Massage: **Add Git and Markdown cheatsheets**
Klicke **Commit changes**
