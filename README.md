# git-marketing-cheatsheets
cheatsheets für Git und Markdown
1.Auf profil gehen
2.Klicke auf **repoitory**
3.Gib einen Namen ein,z.b:**git-markdow-cheatsheets**
4.Wähle public
5.Aktiviere **Add README file**
6.Klicke **Create repository** 
7.Schuche im Internet z.b. nach: **Git Cheat sheet PDF** oder **Markdown Cheat sheet**
8.Öffne dein Repository
9.Klicke **Add file**
10.Klicke **Upload file**
11.Lade die Cheatsheets hoch (PDF oder .md Datei)
12.Unten schreibe bei Commit Massage: **Add Git and Markdown cheatsheets**
13.Klicke **Commit changes**
